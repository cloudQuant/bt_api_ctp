# CTP (China Futures)

CTP exchange plugin for bt_api plugin for `bt_api`.

## Features

- Futures trading support\n- CTP protocol interface

## Installation

```bash
pip install bt_api_ctp
```

## Usage

```python
from bt_api_bt_api_ctp import ctp
# Use the exchange
```

## Architecture

The package follows the standard bt_api plugin structure with feeds, containers, and gateway adapters.
